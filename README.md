## 1. Project Overview

This project focuses on applying Business Intelligence (BI) and data analysis techniques to extract meaningful insights from raw data and support data-driven decision making. The project demonstrates the full analytical workflow, from data understanding and preprocessing to exploratory analysis, visualization, and business interpretation.

The notebook is designed not only to showcase technical skills but also to reflect Business Analyst / Data Analyst thinking, where insights are clearly connected to business objectives.

## 2. Business Objective

The main objectives of this project are:

Understand the structure and quality of the dataset

Identify key patterns, trends, and relationships within the data

Use visualizations to communicate insights effectively

Translate analytical results into actionable business recommendations

## 3. Dataset Description

The dataset used in this project includes multiple variables related to business performance and customer behavior. Key characteristics:

Combination of numerical and categorical features

Presence of missing values and potential outliers

Suitable for correlation analysis and exploratory data analysis (EDA)

Before analysis, the data is carefully reviewed to ensure reliability and consistency.

## 5. Project Workflow
**5.1** Data Loading & Understanding

Import dataset and required libraries

Inspect data structure, data types, and basic statistics

Identify missing values and potential data quality issues

**5.2** Data Cleaning & Preprocessing

Handle missing values appropriately

Standardize data formats

Prepare cleaned data for analysis

**5.3** Exploratory Data Analysis (EDA)

Analyze distribution of key variables

Identify trends and patterns

Explore relationships between variables

**5.4** Correlation Analysis

Build a correlation matrix

Interpret the strength and direction of relationships

Identify variables with strong positive or negative correlations

**5.5** Data Visualization

Use charts and plots to support findings

Ensure visual clarity for business audiences

Highlight key insights through visuals
